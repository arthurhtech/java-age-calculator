# Java Age Calculator

Uma aplicação Java para cálculo de idade, desenvolvida com foco em **Tratamento de Exceções Personalizadas** e documentação via **Javadoc**.

## Objetivo do Projeto

Este projeto foi feito para exercitar conceitos fundamentais de Orientação a Objetos e técnicas de documentação:
* **Exception Handling:** Criação de exceções próprias para validar regras de negócio.
* **Javadoc:** Código documentado para geração automática de páginas HTML.

## Funcionalidades

1.  **Entrada de Dados:** Interface gráfica simples via `JOptionPane` (Swing).
2.  **Validação:** O sistema impede datas inexistentes (ex: mês 13, dia 32) disparando exceções específicas.
3.  **Cálculo de Idade:** Calcula a idade exata baseada na data atual do sistema.
